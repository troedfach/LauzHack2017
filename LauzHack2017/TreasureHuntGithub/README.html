SDK Structure overview

The SDK package contains two directory with the following content:

  • libs/ containing the PixLive SDK library (AAR) that needs to be embedded into your own app project.
  • demos/ containing a sample demo application using PixLive SDK.

Using the demo application

  1. Launch Android Studio. See https://developer.android.com/sdk/index.html for installing Android Studio.
  2. On the welcome dialog, choose import project or go to the File menu and choose New->Import project
  3. Select Existing Projects into Workspace in the General category from the list and click Next
  4. Select the MyARApplication directory inside the demos directory of the SDK
  5. You should now be able to run the application as any other Android application.
  6. Note that you need to change the application package name to your own application package name and add your PixLive SDK License key available in PixLive Maker (PixLive SDK->My Licenses)
